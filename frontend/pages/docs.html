Examples

Practical examples of using ADAM in different scenarios.

JavaScript Example 
const queryADAM = async (question) => {
  const response = await fetch('https://api.adam-ai.com/v1/query', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      query: question,
      parameters: {
        temperature: 0.7,
        max_tokens: 150
      }
    })
  });
  
  return await response.json();
};

// Example usage
queryADAM("Explain quantum computing in simple terms")
  .then(response => console.log(response));
Python Example 
import requests

def query_adam(question):
    url = "https://api.adam-ai.com/v1/query"
    headers = {
        "Authorization": "Bearer YOUR_API_KEY",
        "Content-Type": "application/json"
    }
    data = {
        "query": question,
        "parameters": {
            "temperature": 0.7,
            "max_tokens": 150
        }
    }
    
    response = requests.post(url, headers=headers, json=data)
    return response.json()

# Example usage
response = query_adam("Explain quantum computing in simple terms")
print(response)
