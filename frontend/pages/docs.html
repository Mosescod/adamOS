adamOS Documentation

Latest

adamOS is an AI operating system featuring Adam, a conversational AI with Islamic knowledge foundation and clay metaphors.

Key Features

  • Natural language conversations with Adam AI
  • Theology knowledge integration
  • Contextual memory system
  • Ethical response filtering
  • Conversation history tracking
adamOS is currently in active development. Check GitHub for the latest updates.

Installation Guide

Get adamOS running on your local machine

Prerequisites

Required Software 
- Python 3.8+
- pip
- Git
- Node.js (for frontend development)

Setup Instructions

Clone Repository 
git clone https://github.com/Mosescod/adamOS.git
cd adamOS
Install Python Dependencies 
pip install -r requirements.txt
Run Backend Server 
python app.py
The server should now be running at http://localhost:5000

API Reference

Interact with Adam programmatically

Chat Endpoint

POST /api/chat 
{
  "user_id": "string",
  "message": "string"

Example Response

Success Response 
{
  "status": "success",
  "response": "*shapes clay* Your answer...",
  "user_id": "original-user-id",
  "timestamp": "ISO-8601"

Status Endpoints

Endpoint Method Description
/api/system/status GET Basic system health check
/api/conversation GET Retrieve conversation history

Contributing

How to contribute to adamOS development

Development Workflow

  1. Fork the repository
  2. Create a new branch (git checkout -b feature/your-feature)
  3. Commit your changes
  4. Push to the branch
  5. Create a Pull Request

Code Style

Python Style Guide 
- Follow PEP 8 guidelines
- Type hints for all function signatures
- Docstrings for all public methods
- 4-space indentation
- Single quotes for strings
Please open an issue to discuss major changes before implementing them.
